How the model works

Model architecture

  • Ensemble: XGBoost (40%) + Poisson goals model (60%)
  • Elo ratings built from 49,000+ international matches since 1872
  • Rolling form: last 5 and 10 matches, importance-weighted
  • StatsBomb xG data from 148 World Cup matches
  • Monte Carlo: 10,000 full tournament simulations

Backtest accuracy (leave-one-out, 1994–2022)

XGBoost52.5%
Poisson Goals55.0%
Ensemble (XGB+Poisson)~56%
Random baseline33.3%

Data sources

  • International results 1872–2026 (martj42, Kaggle)
  • FIFA World Cup metadata (abecklas, Kaggle)
  • StatsBomb open data — shot-level xG
  • FIFA rankings (June 2025)
  • 2026 qualified teams & group draw (official FIFA)

Limitations

  • No player-level data (injuries, suspensions)
  • 2026 bracket seeding rules not yet published by FIFA
  • 64-match test set — results vary by ~3–5% per run
  • Football is inherently unpredictable — upsets happen
